== Description ==

The Teaser Slider plugin shows an ajax powered carousel with posts that 
have images attached to them.

== Installation ==

1. Download and unzip the current version of the Teaser Slider plugin.
2. Transfer the entire teaser-slider directory to your `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

The Teaser Slider Carousel can be added to your Wordpress site by 
the following syntax:

* <?php teaserslider();?>

You can also specify the number of posts the slider has by the following 
syntax.

* <?php teaserslider('10');?>

10 being the number of posts viewable in the slider.
